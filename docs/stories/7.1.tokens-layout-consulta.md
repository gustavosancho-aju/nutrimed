## Story

**As a** produto com identidade, **I want** os design tokens do spec e o grid de consulta (transcrição + painel lateral fixo), **so that** a experiência seja calma e profissional (frontend-spec §4/§6).

## Acceptance Criteria

1. Tokens CSS HSL do §6 (cores semânticas dos 4 tipos, acentos por doutor, motion) em globals.css, expostos ao Tailwind 4 via @theme ✅
2. Grid 2 colunas: área principal fluida (transcrição ao vivo) + painel lateral clamp(360px,32vw,460px) ✅
3. Transcrição AO VIVO no cliente: mensagem `transcript` aditiva no protocolo v1 (texto, nunca áudio — §7), reusa <TranscriptPanel> da 2.4 ✅
4. Animações com reduced-motion: zero movimento sem perda de informação ✅
